# 玉米生产力预测平台

一个面向多类玉米预测与模拟服务的 Web 平台公开版仓库，聚焦我负责的平台层能力建设，包括前后端开发、算法服务接入、异步任务调度、基础数据管理、结果预览下载和地图可视化。

本仓库是用于 GitHub 展示的公开安全版本，已移除合作方算法源码、模型运行时、业务数据和其他不适合公开的内容，保留了最能体现工程能力的平台代码。

## 项目亮点

- 基于 `Vue 3 + Vite + FastAPI` 搭建统一平台入口，覆盖算法选择、参数配置、任务提交、结果查看和历史管理
- 按算法模块拆分后端路由，统一封装文件上传、参数读写、结果输出和异常处理流程
- 实现围绕 `task_id` 的异步任务调度机制，支持任务创建、进度查询、取消删除、结果归档和历史记录管理
- 支持基础数据在线维护，包括 JSON/表格上传、在线表格编辑、行列增删改、整表保存与导出
- 支持 `GeoJSON / WMS` 空间结果展示，增强算法结果的可解释性和可视化表达
- 完成配置脱敏与环境化改造，将数据库、地图服务和外部接口从硬编码切换为环境变量配置

## 功能概览

### 1. 平台化算法接入

- 多类玉米算法统一接入到同一前后端平台
- 参数读取、参数更新、默认值重置、任务提交流程统一化
- 结果预览、结果下载、任务历史查看能力统一化

### 2. 任务调度与结果管理

- 长时任务创建与状态跟踪
- 任务进度查询与取消删除
- 结果文件归档与历史记录管理
- 支持大文件上传和计算任务场景

### 3. 基础数据管理

- 分类文件管理
- JSON / 表格文件上传
- 在线表格编辑
- 单元格内联修改
- 行列增删改
- 整表保存与导出

### 4. 空间结果展示

- GeoJSON 结果渲染
- WMS 图层加载
- 地图结果预览
- 统计信息展示

## 系统结构

```text
corn-system (Vue 3 + Vite 前端)
        |
        v
app-py (FastAPI 网关 / 任务调度 / 数据管理)
        |
        v
多类玉米算法服务（私有或外部服务，不包含在公开仓库中）
```

## 目录结构

```text
.
├─ corn-system/    # 前端项目：页面、交互、地图、基础数据管理
├─ app-py/         # 后端项目：API 网关、任务管理、数据管理、用户模型
├─ .gitignore      # 公开版忽略规则，屏蔽私有算法、数据、日志和运行产物
└─ README.md
```

## 快速开始

### 前端

```bash
cd corn-system
cp .env.example .env
npm install
npm run dev
```

### 后端

```bash
cd app-py
cp .env.example .env
pip install -r requirements.txt
python main.py
```

## 环境变量

### 前端

前端环境变量示例位于 `corn-system/.env.example`，主要包括：

- `VITE_APP_TIANDITU_TOKEN`
- `VITE_APP_ARCGIS_WMS_URL`
- `VITE_APP_LEGACY_API_ORIGIN`
- `VITE_DEV_API_TARGET`
- `VITE_DEV_ARCGIS_TARGET`

### 后端

后端环境变量示例位于 `app-py/.env.example`，主要包括：

- `DB_HOST`
- `DB_PORT`
- `DB_USER`
- `DB_PASSWORD`
- `DB_NAME`
- 各算法服务的 `HOST / PORT`

## 公开版说明

本仓库保留的是平台层、交互层和服务接入层代码，不代表完整生产环境部署内容。

### 已保留

- 前端页面与交互逻辑
- FastAPI 网关与任务调度代码
- 基础数据管理与地图展示能力
- 环境变量示例与公开版仓库说明

### 未开源

- 合作方或第三方提供的算法源码

- 业务数据、样例数据和输出结果

## 这个仓库体现的工程能力

这个公开仓库重点体现的能力包括：

- 平台化思维：将分散的算法能力整合为统一平台
- 后端工程能力：接口拆分、任务调度、文件处理、配置管理
- 前后端协同能力：参数配置、结果展示、任务历史、地图联动

## 说明

- 当前公开仓库以展示平台工程能力为主
- 某些算法页面依赖私有服务，因此公开版更适合展示平台结构与交互流程
- 如需二次整理或继续公开，请始终在当前项目根目录下操作，以确保顶层 `.gitignore` 生效
